Política de Privacidad de la aplicación KVP Launcher (Interface for the Elderly)

Esta política de privacidad describe cómo nuestra aplicación recopila, utiliza y protege la información de los usuarios. Hemos diseñado esta aplicación como un software gratuito destinado a personas mayores.

1. Recopilación y uso de datos personales

Nuestra aplicación requiere acceso a ciertas funciones del dispositivo con el único propósito de garantizar las funciones básicas del lanzador (launcher):

  • SMS (SEND_SMS): Se utiliza exclusivamente para enviar un mensaje de texto de emergencia a contactos predefinidos cuando se activa el botón de SOS.
  • Contacts (READ_CONTACTS): Se utiliza exclusivamente para seleccionar y mostrar los nombres de los cuidadores en la pantalla de inicio para realizar llamadas rápidas. Las llamadas perdidas y los mensajes entrantes se procesan de forma segura mediante el acceso estándar a las notificaciones, sin leer los registros del sistema.
  • Cámara (CAMERA): Se utiliza exclusivamente para el correcto funcionamiento de la función de linterna.
  • Visibilidad de paquetes (QUERY_ALL_PACKAGES): Se utiliza exclusivamente para mostrar la lista completa de aplicaciones instaladas en la pantalla de inicio y permitir al usuario iniciarlas.
  • Acceso a notificaciones (NotificationListenerService): Se utiliza solo para interceptar y mostrar las notificaciones entrantes (como Viber o SMS) en un cuadro de diálogo ampliado y accesible directamente en la pantalla principal.

2. Transferencia de datos a terceros

Nosotros no recopilamos, almacenamos, rastreamos ni transferimos ninguno de sus datos personales, contenido de mensajes o historial de llamadas a servidores externos ni a terceras partes. Todas las operaciones de datos se realizan de forma estrictamente local en su dispositivo.

3. Cambios en la política de privacidad

Podemos actualizar nuestra Política de Privacidad periódicamente. Los cambios entrarán en vigor a partir del momento de su publicación en esta página.
