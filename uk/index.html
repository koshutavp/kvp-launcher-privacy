Політика конфіденційності додатка KVP Launcher (Interface for the Elderly)

Ця політика конфіденційності описує, як наш додаток збирає, використовує та захищає інформацію користувачів. Ми створили цей додаток як безкоштовне програмне забезпечення для літніх людей.

1. Збір та використання персональних даних

Наш додаток потребує доступу до певних функцій пристрою виключно для забезпечення базової функціональності лаунчера:

  • SMS (SEND_SMS): Використовується виключно для надсилання екстреного текстового повідомлення заздалегідь визначеним контактам при активації кнопки SOS.
  • Contacts (READ_CONTACTS): Використовується виключно для вибору та відображення імен опікунів на головному екрані для швидкого набору. Пропущені виклики та вхідні повідомлення безпечно обробляються через стандартний доступ до сповіщень, без зчитування системних журналів.
  • Камера (CAMERA): Використовується виключно для роботи функції ліхтарика.
  • Видимість пакетів (QUERY_ALL_PACKAGES): Використовується виключно для відображення повного списку встановлених додатків на головному екрані та надання користувачеві можливості їх запускати.
  • Доступ до сповіщень (NotificationListenerService): Використовується тільки для перехоплення та відображення вхідних сповіщень (наприклад, Viber або SMS) у збільшеному, доступному діалоговому вікні на головному екрані.

2. Передача даних третім особам

Ми не збираємо, не зберігаємо, не відстежуємо та не передаємо жодні ваші персональні дані, вміст повідомлень чи історію дзвінків на сторонні сервери чи третім особам. Усі операції з даними відбуваються локально на вашому пристрої.

3. Зміни в політиці конфіденційності

Ми можемо періодично оновлювати нашу Політику конфіденційності. Зміни вступають в силу з моменту їх публікації на цій сторінці.
