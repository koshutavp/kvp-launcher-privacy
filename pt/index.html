Política de Privacidade do aplicativo KVP Launcher (Interface for the Elderly)

Esta política de privacidade descreve como o nosso aplicativo coleta, usa e protege as informações dos usuários. Desenvolvemos este aplicativo como um software gratuito destinado a idosos.

1. Coleta e uso de dados pessoais

O nosso aplicativo requer acesso a certas funções do dispositivo exclusivamente para garantir a funcionalidade básica do launcher:

  • SMS (SEND_SMS): Usado exclusivamente para enviar uma mensagem de texto de emergência para contatos predefinidos quando o botão SOS é ativado.
  • Contacts (READ_CONTACTS): Usado exclusivamente para selecionar e exibir os nomes dos cuidadores na tela inicial para discagem rápida. Chamadas perdidas e mensagens recebidas são processadas com segurança através do acesso padrão às notificações, sem a leitura de logs do sistema.
  • Câmera (CAMERA): Utilizado estritamente para o funcionamento da função de lanterna.
  • Visibilidade de pacotes (QUERY_ALL_PACKAGES): Usado exclusivamente para exibir a lista completa de aplicativos instalados na tela inicial e permitir que o usuário os inicie.
  • Acesso a notificações (NotificationListenerService): Usado apenas para interceptar e exibir notificações recebidas (como Viber ou SMS) em uma caixa de diálogo ampliada e acessível na tela principal.

2. Compartilhamento de dados com terceiros

Nós não coletamos, armazenamos, rastreamos ou transmitimos nenhum dos seus dados pessoais, conteúdo de mensagens ou histórico de chamadas para servidores externos ou terceiros. Todas as operações com os dados são realizadas localmente no seu dispositivo.

3. Alterações nesta política de privacidade

Podemos atualizar a nossa Política de Privacidade periodicamente. As alterações entram em vigor a partir do momento da sua publicação nesta página.
